Pointers

Posted on Aug 11, 2021

Definition

  • define an integer and reserves a location in memory for this specific variable
    • take an integer #!cpp pointer:
int *pointer
  • this defines an integer #!cpp pointer such that its content is the address of another place in memory
int var
var = 10
pointer = &var
  • this defines an integer #!cpp var, and #!cpp &var is the address of a specific place in memory where the integer is stored
    • it also points the variable #!cpp pointer to #!cpp var
  • so that when checking out the value #!cpp *pointer*, we get the content of #!cpp var, which is 10

With arrays

  • pointers can also be appoined to arrays so that using #!cpp (pointer + 1) points to certain array element #!cpp matr[1].
