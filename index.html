This is my personal site. Here, I write things of all kinds. It's primarily to document what I learn in the big world of tech and privacy, and other things such as music, films, hardware. My opinions are my own.

It is generated by Jekyll and hosted on Github (WIP to host it libre). There is no JS, it's in pure HTML & CSS. The Jekyll template is available here. This website uses the gruvbox theme.
