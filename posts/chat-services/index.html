Chat services

2021-07-23
tech

Here’s a qualm: there are simply too many chat services, it’s splitting my brain

  • discord as primary communication with my boyfriend, some of my friends, and most of popular public servers
    • discord + e2ee @ browser for main
    • discord –> insular @ android for main
    • discord electron + betterdiscord integrations @ linux for alt, only for calling my boyfriend
  • telegram with my parents & lingering privacy-centric groups, or android modding groups
    • nekogram x @ android
    • telegram k web @ browser
  • matrix (with libera bridge) for most public groups that are worthy enough to use the protocol, mainly privacy-focussed group and tech-related
    • fluffychat @ android
    • mirage @ linux
  • whatsapp for absolute normies (uni groups, housemates, most irl boomers)
    • whatsapp –> insular @ android
    • whatsapp web @ browser but cookies are not saved
  • messenger/facebook/instagram for some old friends, but I probably open those websites once a year at this point
    • barinsta –> insular @ android with chat function
    • web versions @ browser but again, I simply avoid using it
