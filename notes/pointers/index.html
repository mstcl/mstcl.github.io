Pointers

Posted on Aug 11, 2021

Definition

  • define an integer and reserves a location in memory for this specific variable
    • take an integer pointer:
int *pointer
  • this defines an integer pointer such that its content is the address of another place in memory
int var
var = 10
pointer = &var
  • this defines an integer var, and &var is the address of a specific place in memory where the integer is stored
    • it also points the variable pointer to var
  • so that when checking out the value *pointer*, we get the content of var, which is 10

With arrays

  • pointers can also be appointed to arrays so that using (pointer + 1) points to certain array element matr[1].
